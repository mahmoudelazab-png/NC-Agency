# NC Agency OS - System Architecture


## 1. Overview

NC Agency OS will be built as a modern web application with scalable architecture.


## 2. System Layers


### Frontend Layer

Responsible for user interface.

Technology:

- React.js
- Next.js
- Tailwind CSS


Responsibilities:

- Dashboard
- Client management screens
- Task management
- Reports
- User interface


---


### Backend Layer

Responsible for business logic and APIs.

Technology:

- Node.js
- Express.js


Responsibilities:

- Authentication
- User permissions
- Data processing
- API management


---


### Database Layer

Responsible for storing system data.

Technology:

- PostgreSQL


Main entities:

- Users
- Clients
- Leads
- Projects
- Tasks
- Campaigns
- Finance


---


## 3. System Flow


User

↓

Frontend Application

↓

Backend API

↓

Database

## 4. Security
