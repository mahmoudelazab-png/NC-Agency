# NC Agency OS - Software Requirements Specification

## 1. Project Overview

NC Agency OS is a complete management system designed for NC Marketing Agency.

The system will manage:
- Clients
- Sales
- Projects
- Tasks
- Content Production
- Advertising Campaigns
- Team Management
- Finance
- Reports


## 2. User Roles

### Admin
Full control over the system.

### Account Manager
- Manage clients
- Track communication
- Create tasks

### Media Buyer
- Manage campaigns
- Track budgets
- Analyze performance

### Content Creator
- Manage content calendar
- Upload designs
- Track approvals

### Designer
- Manage design tasks
- Upload files

### Finance
- Manage invoices
- Track payments
